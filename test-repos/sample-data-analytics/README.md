# Sample Data Analytics (Python 2)

A legacy data analytics tool for testing migration.

## Python 2 Patterns Used:
- `print` statements
- `__builtin__` module (should be `builtins`)
- `cPickle` module
- `cStringIO` module
- `raw_input()` function
- `dict.iteritems()` method
- `reduce()` built-in (should be `functools.reduce`)
- `filter()` returning list
- `map()` returning list
- Integer division without `//`
- `%` string formatting
- Old-style string formatting in loops
