# Sample Web Scraper (Python 2)

A legacy web scraper with Python 2 patterns for testing migration.

## Python 2 Patterns Used:
- `print` statements
- `urllib2` module
- `urlparse` module  
- `HTMLParser` module
- `cPickle` module
- `StringIO` module
- `raw_input()` function
- Old-style exception handling (`except Error, e`)
- `%` string formatting
