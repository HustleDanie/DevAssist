# Sample Database Utility (Python 2)

A legacy database management utility for testing migration.

## Python 2 Patterns Used:
- `print` statements
- `anydbm` module (should be `dbm`)
- `cPickle` module
- `UserDict.UserDict` class
- `thread` module (should be `_thread`)
- `Queue` module (should be `queue`)
- `raw_input()` function
- `dict.has_key()` method
- `dict.iteritems()` method
- Old-style exception handling
- `%` string formatting
