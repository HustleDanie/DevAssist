# Legacy Python 2 Application

This is a sample Python 2 application for testing DevAssist migration tool.

## Features
- Calculator module with basic operations
- User management system
- Data processing utilities
- Simple Flask web API

## Testing Migration

Use this repository to test:
1. **Python 2 → Python 3** migration
2. **Flask → FastAPI** migration

## Known Python 2 Patterns
- `print` statements (not functions)
- `xrange()` usage
- `raw_input()` for user input
- `dict.iteritems()`, `dict.iterkeys()`, `dict.itervalues()`
- `except Exception, e:` syntax
- `unicode()` function
- Integer division behavior
