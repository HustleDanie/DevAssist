# Sample File Processor (Python 2)

A legacy file processing utility for testing migration.

## Python 2 Patterns Used:
- `print` statements
- `ConfigParser` module (should be `configparser`)
- `cStringIO` module
- `raw_input()` function
- Old-style exception handling
- `%` string formatting
- `dict.iteritems()` method
- `filter()` with lambda returning list
